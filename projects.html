MangoTango

MangoTango is a smart pest-monitoring system for mango farmers. It uses AI to detect pests by analyzing images of leaves and fruit. The system provides timely treatment advice, tracks farm activities, sends weather-based alerts, and helps farmers manage infestations early and sustainably.

MangoTango Backend

The MangoTango backend is a lightweight cloud server focused on data and user management. It stores farmer profiles, pest history, treatment logs, and farm activity data in a secure database. The backend also handles sending weather-based pest alerts and syncing user data across devices, while the machine learning model runs locally on the mobile application.
